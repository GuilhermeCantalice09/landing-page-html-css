💻 Meu Primeiro Site (HTML + CSS)

Este foi o meu primeiro projeto completo de desenvolvimento web, criado com o objetivo de testar e consolidar meus conhecimentos em HTML e CSS.

🚀 Sobre o projeto

O projeto consiste em um site completo desenvolvido do zero, onde trabalhei toda a estrutura, organização e estilização da página.

O layout da landing page foi inspirado em uma referência existente, com foco em praticar a reprodução de interfaces reais — algo comum no aprendizado de front-end.

🛠️ Tecnologias utilizadas
HTML5
CSS3
📚 O que eu aprendi
Estruturação de páginas com HTML
Organização de arquivos
Estilização com CSS
Uso de layout (Flexbox)
Construção de um site completo do zero
